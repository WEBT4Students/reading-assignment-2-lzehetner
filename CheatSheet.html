Reading Assignment 2: WWW and html

HTML

  1. Name the components of html elements properly
    The components of html elements are Start-Tag, maybe an Attribute, element content and End-Tag

  2. Identify void elements and how they are denoted
    HTML elements with no content are called void elements. It is closed in its Start-Tag, seen by the slash at the end. The slash can also be omitted(=weggelassen). An example is "br"

  3. Identify attributes
    Attributes provide additional information about elements. All HTML elements can have them. Attributes are always specified in the start tag

  4. Write down the correct DOCTYPE declaration for html 5
    The declaration is not an HTML tag. It is an "information" to the browser about what document type to expect("!DOCTYPE html")

  5. Write down syntactically correct statements for the following elements
    • Article (article)
      • Potential sources for the "article" element: Forum -, Blog Post and News Story
    • Body (body)
      • The "body" element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
      Note: There can only be on body in an HTML document
    • line break (br/)
      • The "br" tag is an empty tag which means that it has no end tag
    • Headings (h1, ...)
      • "h1" defines the most important heading. "h6" defines the least important heading
    • Section with meta-information (head)
      • The "head" element is a container for metadata (data about data) and is placed between the "html" tag and the "body" tag
    • Top level element (html)
      • The "html" tag represents the root of an HTML document
    • Image named Team.jpg (img src="Team.jpg")
      • Images are not technically inserted into a web page; images are linked to web pages. The "img" tag creates a holding space for the referenced image
    • Paragraph (p)
      • The "p" tag defines a paragraph
    • Section (section)
      • The "section" tag defines a section in a document.
    • Label appearing in the title bar (title)
      • The "title" tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab The "title" tag is required in HTML documents
    • Link to other pages (a)
      • The "a" tag defines a hyperlink, which is used to link from one page to another
    • Comment
      • "!--...--"
    • Head with title and meta elements
      • The HTML "head" element is a container for the following elements: "title", "style", "meta", "link", "script", and "base".

CSS

  1. Name the components of a css rule properly
    The selector points to the HTML element you want to style
    The declaration block contains one or more declarations separated by semicolons
    Each declaration includes a CSS property name and a value, separated by a colon
    ultiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces

  2. Use combinators properly: direct child, descendant, adjacent sibling, sibling
    • descendant selector (space) The descendant selector matches all elements that are descendants of a specified element
      Example: div p {background-color: yellow;}

    • child selector (>) The child selector selects all elements that are the children of a specified element
      Example: div ">" p {background-color: yellow;}

    • adjacent sibling selector (+) The adjacent sibling selector is used to select an element that is directly after another specific element
      Sibling elements must have the same parent element, and "adjacent" means "immediately following"
      Example: div "+" p {background-color: yellow;}

    • general sibling selector (~) The general sibling selector selects all elements that are next siblings of a specified element
      Example: div "~" p {background-color: yellow;}


  3. Specify colors by color names and in hexadecimal notation
    For example, to specify white color you can write #FFFFFF or #ffffff

  4. Declarations for color, background color
    • color names: background-color: green;
    • hex values: background-color: #46f2e1;
    • rgb(): background-color: rgb(25, 31, 52);
    • rgba() to add transparency: background-color: rgb(25, 31, 52, 0.8);
    • hsl(): background-color: hsl(227, 33%, 22%);
    • hsla() to add transparency: background-color: hsla(227, 33%, 22%, 50%);
    • hwb(): background-color: hwb(222 22% 9%);;
    • lab(): background-color: lab(74.19% -20.68 -29.03);
    • lch(): background-color: lch(59.65% 68.2 22.81);
    • color(): background-color: color(display-p3 0.75 0.5 0.523);

  5. Pseudo classes for the "a" element
    a:link => unvisited link
    a:visited => visited link
    a:hover => mouse over link
    a:active => selected link

  6. Declaration for background image
    • background-color

    • background-image

    • background-repeat

    • background-attachment

    • background-position

    Example: background: #ffffff url("img_tree.png") no-repeat right top;

  7. Declaration for text alignment (left, right, center, justify)
    div {text-align: right|left|center|justify}

  8. Declaration for text decoration (overline, underline, line-through)
    The text-decoration property specifies the decoration added to text, and is a shorthand property for:
    • text-decoration-line (required)

    • text-decoration-color

    • text-decoration-style

    • text-decoration-thickness

    Example: h3{text-decoration: overline;}

  9. Declaration for text transformation (uppercase, lowercase, capitalize)
    The text-transform property controls the capitalization of text.
    div {text-transform: none|capitalize|uppercase|lowercase;}

  10. Identify the difference between serif and sans-serif fonts
    Serif: fonts have a small stroke at the edges of each letter. They create a sense of formality and elegance
    Sans-Serif: fonts have clean lines (no small strokes attached). They create a modern and minimalistic look.

  11. Declarations for font families
    The font-family property can hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font
    Example: p {font-family: "Times New Roman", Times, serif;}

  12. Declarations for font style normal and italic
    normal: The browser displays a normal font style. This is default
    italic The browser displays an italic font style
    Example: p {font-style: normal|italic;}

  13. Declarations for font sizes
    The font-size property sets the size of a font.
    Example: div {font-size: medium|xx-small|x-small|small|large|x-large|xx-large|smaller|larger|length|initial|inherit;}

  14. Declarations for font weights
    The font-weight property sets how thick or thin characters in text should be displayed
    Example: p {font-weight: normal|bold|bolder|lighter|number|initial|inherit;; }
